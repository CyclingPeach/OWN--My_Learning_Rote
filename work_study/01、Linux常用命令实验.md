# 1、基本命令
## （1） 关机和重启
|序号|命令|作用|备注|
|:-:|:-|:-:|-|
|1|`shutdown -h now`|立刻关机|
|2|`shutdown -h 5`|5分钟后关机|
|3|`poweroff`|立刻关机|
|4|`shutdown -r now`|立刻重启|
|5|`shutdown -r 5`|5分钟后重启|
|6|`reboot`|立刻重启|
